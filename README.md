# 献立管理アプリ

日々の献立を記録・管理できるWebアプリです。

## URL
https://menu-app-neon.vercel.app/

---

## 機能

- 献立の新規登録
- 献立の編集・削除
- 献立一覧表示（週間表示）
- カテゴリー別表示
- 画像アップロード

---

## 使用技術

- React
- TypeScript
- Vite
- Supabase（DB / Storage）
- React Router
- Vercel（デプロイ）

---

## 工夫した点

- Enterキーで入力欄を追加、Backspaceで削除できるUIを実装
- useRefを使って入力フォーカス制御
- 週間表示・月別表示など日付ロジックを実装
- 画像アップロード機能を実装（Supabase Storage）
